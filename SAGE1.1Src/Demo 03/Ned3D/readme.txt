Keyboard commands for this demo are:

  Esc   - Quit
  ~     - Toggle show/hide the console
            Type 'help -a' (no quotes) when the console is visible to get a
            list of commands supported by the console.
  
  Enter - Moves the plane forward
  W     - Dives
  S     - Climbs
  A     - Turns left
  D     - Turns right
